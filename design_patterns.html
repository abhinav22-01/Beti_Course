What are Design Patterns?

Design patterns are typical solutions to common problems in software design. They are blueprints for solving issues that arise in coding.

Letâ€™s start with a basic pattern: Singleton Pattern.

A Singleton ensures that a class has only one instance and provides a global point of access to it.

Exercise: Implement Singleton in Python

class Singleton:
    _instance = None

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super(Singleton, cls).__new__(cls)
        return cls._instance

singleton1 = Singleton()
singleton2 = Singleton()

print(singleton1 == singleton2)  # True
